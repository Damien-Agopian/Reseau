#   APPLICATION D'APPRENTISSAGE DE LANGUES   #
#============================================#

Auteurs : Damien AGOPIAN, Chloé DAUDIER, Solène GONTHIER

Libraries nécessaires :
* threading
* socket
* Tkinter
* time
* PIL de pillow
* random
* sys
Normalement la majorité des libraries sont déjà installées sur les machines.

Documents :
* serveur.py = serveur
* interface.py = client et interface graphique
* dico.py = dictionnaire qui contint les hiragana, les katakana et les signes (langage 
  des signes)
* Images utilisées dans l'interface graphique 

Lancement de l'application :
* Pour utiliser cette application, commencez par lancer un serveur (python3 serveur.py), 
  puis le ou les clients (python3 interface.py). Pour le serveur vous pouvez préciser le 
  port sur lequel vous voulez travailler (python3 serveur.py 8001). Du côté client vous
  pouvez préciser l'adresse IP et le port (python3 interface.py "localhost" 8001). 
* A partir de l'interface vous pourrez accéder à différents exercices grâce au menu.
